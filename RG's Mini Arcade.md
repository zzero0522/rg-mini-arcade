# 這是 RG's Mini Arcade 規劃書

## 需求
- 實現幾個上班或平時無聊時，純前端的小遊戲
- 介面必須有一般款與融入上班款(IDE風格)
- 全部純前端，避免任何資安問題與儲存議題
- 放在 GitHub，以 GitHub Pages 形式公開
- 使用 localStorage 保存最高分數與設定（不涉及後端）
- 響應式設計，支援手機與桌面

## 技術規劃
- 純 HTML / CSS / JavaScript（無框架依賴）
- 主題切換：一般模式 / IDE 模式
- 統一遊戲選單入口頁面
